# Swami Math Classes Online Learning Platform

Monorepo skeleton aligned to finalized system documentation.

## Components
- `backend/` Firebase Cloud Functions API backend
- `student_app/` Flutter Android app skeleton
- `admin_panel/` Flutter Windows admin app skeleton
- `infrastructure/` Firebase and CI/CD configs
- `environments/` Environment variable templates
- `scripts/` Deployment helper scripts
- `docs/` Documentation placeholders grouped by domain
